# Story 1.5: Application Shell, Navigation and Tab System

## Status: Done

## Story

**As a** developer,
**I want** a polished app shell with sidebar, tabbed content, breadcrumbs, and status bar,
**so that** all features have consistent navigation.

## Acceptance Criteria

1. Main window has a three-zone layout: sidebar (left), content area (center), status bar (bottom).
2. Sidebar displays grouped module icons with labels and supports collapsing to icon-only mode.
3. Tab system supports opening, closing, closing all, reordering (drag), and overflow scrolling.
4. Breadcrumbs show the current navigation path and allow clicking to navigate back.
5. Status bar displays: connected domain name, domain controller, current permission level, and connection status indicator.
6. `INavigationService` provides programmatic navigation, tab management, and breadcrumb tracking.
7. Keyboard shortcuts are implemented for common navigation actions.
8. Window state (size, position, maximized) is persisted and restored on next launch.
9. Layout is responsive - sidebar collapses automatically at narrow widths.

## Dev Notes

### INavigationService Interface

```csharp
namespace DSPanel.Services.Navigation;

public interface INavigationService
{
    IReadOnlyList<TabItem> OpenTabs { get; }
    TabItem? ActiveTab { get; }
    IReadOnlyList<BreadcrumbSegment> Breadcrumbs { get; }

    void NavigateTo(string moduleId, string? entityId = null);
    TabItem OpenTab(string title, string moduleId, string? icon = null);
    void CloseTab(TabItem tab);
    void CloseAllTabs();
    void ActivateTab(TabItem tab);
    void MoveTab(int fromIndex, int toIndex);
}
```

### Tab Data Model

```csharp
namespace DSPanel.Services.Navigation;

public sealed class TabItem : ObservableObject
{
    public required string Id { get; init; }
    public required string Title { get; set; }
    public required string ModuleId { get; init; }
    public string? Icon { get; init; }
    public bool IsPinned { get; init; }
    public object? Content { get; set; } // The ViewModel for this tab
}

public sealed class BreadcrumbSegment
{
    public required string Label { get; init; }
    public required string NavigationTarget { get; init; }
}
```

### MainViewModel Structure

```csharp
public partial class MainViewModel : ObservableObject
{
    private readonly INavigationService _navigation;
    private readonly IPermissionService _permissions;
    private readonly IDirectoryProvider _directoryProvider;

    [ObservableProperty] private bool _isSidebarExpanded = true;
    [ObservableProperty] private TabItem? _selectedTab;

    public ObservableCollection<TabItem> Tabs { get; } = [];
    public ObservableCollection<SidebarModule> SidebarModules { get; } = [];

    // Domain info for status bar
    public string? DomainName => _directoryProvider.DomainName;
    public bool IsConnected => _directoryProvider.IsConnected;
    public PermissionLevel PermissionLevel => _permissions.CurrentLevel;

    [RelayCommand]
    private void ToggleSidebar() => IsSidebarExpanded = !IsSidebarExpanded;

    [RelayCommand]
    private void CloseTab(TabItem tab) => _navigation.CloseTab(tab);

    [RelayCommand]
    private void CloseAllTabs() => _navigation.CloseAllTabs();
}
```

### Sidebar Module Definitions

```csharp
public sealed class SidebarModule
{
    public required string Id { get; init; }
    public required string Label { get; init; }
    public required string IconKey { get; init; } // Key in IconPaths.xaml
    public required string Group { get; init; } // "Directory", "Tools", "Settings"
    public PermissionLevel RequiredLevel { get; init; } = PermissionLevel.ReadOnly;
}
```

Initial modules for Epic 1:
- Group "Directory": User Lookup, Computer Lookup, Group Lookup
- Group "Tools": (empty until later epics)
- Group "Settings": Theme Toggle, About

### Window State Persistence

Store window state as a JSON file in the application's local data folder:

```csharp
var settingsPath = Path.Combine(
    Environment.GetFolderPath(Environment.SpecialFolder.LocalApplicationData),
    "DSPanel", "window-state.json");
```

```json
{
  "left": 100,
  "top": 50,
  "width": 1280,
  "height": 800,
  "windowState": "Normal",
  "sidebarExpanded": true
}
```

On restore, validate that the saved position is within current screen bounds. If not, reset to defaults.
