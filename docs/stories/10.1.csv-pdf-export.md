# Story 10.1: CSV and PDF Export

## Status: Draft

## Story

**As a** DSPanel operator,
**I want** to export any list or table view to CSV or PDF,
**so that** I can share data with stakeholders and maintain offline records.

## Acceptance Criteria

1. An export button is available on all list and table (React data table) views throughout the application.
2. CSV export produces a UTF-8 file with BOM, with configurable delimiter (comma, semicolon, tab).
3. PDF export includes a title header, generation timestamp, page numbers, and properly formatted table data.
4. A file save dialog opens with a suggested filename based on the view name and current date.
5. A progress indicator is shown for large exports (1000+ rows).
6. Unit tests cover CSV formatting, PDF generation, and the export service interface.

## Dev Notes

### csv crate Configuration

```typescript
var config = new CsvConfiguration(i18n locale.InvariantCulture)
{
    Delimiter = options.Delimiter,
    Encoding = new UTF8Encoding(encoderShouldEmitUTF8Identifier: true),
    HasHeaderRecord = options.IncludeHeaders
};
```

### printpdf crate Document Structure

```typescript
Document.Create(container =>
{
    container.Page(page =>
    {
        page.Header().Text(title).FontSize(16).Bold();
        page.Content().Table(table => { /* columns and rows */ });
        page.Footer().AlignCenter().Text(x =>
        {
            x.Span("Page ");
            x.CurrentPageNumber();
            x.Span(" of ");
            x.TotalPages();
        });
    });
}).GeneratePdf(filePath);
```
