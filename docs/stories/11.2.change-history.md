# Story 11.2: AD Change History Timeline

## Status: Done

## Story

**As a** DSPanel operator,
**I want** to view the change history of any AD object as a timeline,
**so that** I can see what attributes changed, when, and what the previous values were.

## Acceptance Criteria

1. A "History" tab is available on every AD object detail view (user, group, computer, OU).
2. The timeline displays attribute changes with: attribute name, timestamp, originating DC, old value, and new value.
3. Data is sourced from the `msDS-ReplAttributeMetaData` attribute on each object.
4. The timeline is sortable by date (newest first by default) and filterable by attribute name.
5. Multi-value attributes (e.g., member) show a visual diff highlighting added and removed values.
6. Unit tests cover metadata parsing, timeline construction, and diff visualization logic.

## Dev Notes

### msDS-ReplAttributeMetaData Format

The attribute is a constructed attribute returning XML:

```xml
<DS_REPL_ATTR_META_DATA>
  <pszAttributeName>displayName</pszAttributeName>
  <dwVersion>3</dwVersion>
  <ftimeLastOriginatingChange>2026-02-15T10:30:00Z</ftimeLastOriginatingChange>
  <uuidLastOriginatingDsaInvocationID>GUID</uuidLastOriginatingDsaInvocationID>
  <usnOriginatingChange>12345</usnOriginatingChange>
</DS_REPL_ATTR_META_DATA>
```

Note: This metadata provides the version number and last change timestamp but does not contain historical values. To show before/after values, the application would need to compare with stored snapshots from `SnapshotService` trait (if available).
