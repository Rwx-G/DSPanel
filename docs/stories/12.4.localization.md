# Story 12.4: Localization Support

## Status: Draft

## Story

**As a** DSPanel user,
**I want** the application to be available in English and French with proper locale formatting,
**so that** I can use the application in my preferred language.

## Acceptance Criteria

1. All user-facing strings are extracted to i18n JSON resource files.
2. English is the default language with `en.json`.
3. A complete French translation is provided in `fr.json`.
4. A language selector in Settings allows switching between available languages (restart required).
5. Date, number, and currency formatting respect the selected locale's `i18n locale`.
6. Developer documentation explains how to add a new language.
7. Unit tests verify that all resource keys exist in both language files and that locale switching works.

    - [ ] Replace hardcoded strings with resource references (`useTranslation` hook in TSX, `t("key")` in TypeScript)

## Dev Notes

### Resource File Organization

```
src/locales/
  en.json          (English - default)
  fr.json          (French)
```

### TSX Resource Binding

```tsx
const { t } = useTranslation();
<span>{t("passwordReset.title")}</span>;
```

### TypeScript Resource Access

```typescript
const message = t("passwordReset.success");
```

### i18n Setup (App.tsx)

```typescript
import i18n from "i18next";
import { initReactI18next } from "react-i18next";

const language = settings.appearance.language;
i18n.use(initReactI18next).init({
    lng: language,
    resources: { en, fr },
    fallbackLng: "en",
});
```

### Naming Convention for Resource Keys

Use `{View}_{Element}` pattern:

- `MainWindow_Title`
- `PasswordReset_Button`
- `SecurityDashboard_AlertCount`
- `Common_Save`, `Common_Cancel`, `Common_Error`

### File Layout

- `src/locales/en.json`
- `src/locales/fr.json`

### Testing

- Use reflection to compare resource keys between `i18n JSON` files.
- Use standard Rust or TypeScript assertions for readable assertions.
- Test file: `src-tauri/tests/Localization/ResourceCompletenessTests.rs`
- Test file: `src-tauri/tests/Localization/CultureSwitchingTests.rs`
