# Story 12.4: Localization Support

## Status: Draft

## Story

**As a** DSPanel user,
**I want** the application to be available in English and French with proper locale formatting,
**so that** I can use the application in my preferred language.

## Acceptance Criteria

1. All user-facing strings are extracted to `.resx` resource files.
2. English is the default language with `Resources.resx`.
3. A complete French translation is provided in `Resources.fr.resx`.
4. A language selector in Settings allows switching between available languages (restart required).
5. Date, number, and currency formatting respect the selected locale's `CultureInfo`.
6. Developer documentation explains how to add a new language.
7. Unit tests verify that all resource keys exist in both language files and that locale switching works.

## Tasks / Subtasks

- [ ] Task 1: Extract all strings to `Resources.resx` (AC: #1, #2)
  - [ ] Audit all XAML files for hardcoded strings

  - [ ] Replace hardcoded strings with resource references (`x:Static` in XAML, `Resources.Key` in C#)

## Dev Notes

### Resource File Organization

```
Properties/
  Resources.resx          (English - default)
  Resources.fr.resx       (French)
  Resources.Designer.cs   (auto-generated)
```

### XAML Resource Binding

```xml
<TextBlock Text="{x:Static properties:Resources.PasswordReset_Title}" />
```

### C# Resource Access

```csharp
var message = Properties.Resources.PasswordReset_Success;
```

### CultureInfo Setup (App.xaml.cs)

```csharp
var language = settingsService.Load().Appearance.Language;
var culture = new CultureInfo(language);
Thread.CurrentThread.CurrentUICulture = culture;
Thread.CurrentThread.CurrentCulture = culture;
FrameworkElement.LanguageProperty.OverrideMetadata(
    typeof(FrameworkElement),
    new FrameworkPropertyMetadata(XmlLanguage.GetLanguage(culture.IetfLanguageTag)));
```

### Naming Convention for Resource Keys

Use `{View}_{Element}` pattern:
- `MainWindow_Title`
- `PasswordReset_Button`
- `SecurityDashboard_AlertCount`
- `Common_Save`, `Common_Cancel`, `Common_Error`

### File Layout

- `Properties/Resources.resx`
- `Properties/Resources.fr.resx`

### Testing

- Use reflection to compare resource keys between `.resx` files.
- Use `FluentAssertions` for readable assertions.
- Test file: `src/DSPanel.Tests/Localization/ResourceCompletenessTests.cs`
- Test file: `src/DSPanel.Tests/Localization/CultureSwitchingTests.cs`
