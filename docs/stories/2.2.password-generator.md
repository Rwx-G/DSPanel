# Story 2.2: Secure Password Generator

## Status: Done

## Story

**As a** HelpDesk operator,
**I want** a password generator with configurable criteria and breach detection,
**so that** I can create strong, uncompromised passwords for users.

## Acceptance Criteria

1. `PasswordGenerator` produces passwords matching configurable criteria: length, uppercase, lowercase, digits, special characters, and ambiguous character exclusion.
2. `HibpClient` checks generated passwords against the HIBP Pwned Passwords API using k-anonymity (only the first 5 characters of the SHA1 hash are sent).
3. If a password is found in HIBP, it is automatically regenerated (up to a configurable retry limit).
4. A standalone password generator tool is available as a page/tab in DSPanel, usable by all permission levels.
5. The generator is also integrated into the `PasswordResetDialog` from Story 2.1.
6. When the HIBP API is unreachable, the generator falls back to offline mode and displays a warning that breach checking is unavailable.
7. Unit tests cover generation logic, HIBP integration, offline fallback, and retry behavior.

## Dev Notes

### HIBP k-Anonymity Protocol

The Have I Been Pwned API uses k-anonymity so the full password hash is never transmitted:

1. Compute `SHA1(password)` - e.g., `5BAA61E4C9B93F3F0682250B6CF8331B7EE68FD8`
2. Send GET request to `https://api.pwnedpasswords.com/range/5BAA6`
3. Response contains suffixes like `1E4C9B93F3F0682250B6CF8331B7EE68FD8:3861493`
4. Match the suffix portion of the local hash against returned suffixes
5. If matched, the count indicates how many breaches contain this password

### PasswordGenerator Options

```typescript
pub struct PasswordOptions(
    int Length = 16,
    bool IncludeUppercase = true,
    bool IncludeLowercase = true,
    bool IncludeDigits = true,
    bool IncludeSpecial = true,
    bool ExcludeAmbiguous = false);
```

Ambiguous characters: `0 O o l I 1 | ` (backtick). When `ExcludeAmbiguous` is true, these are removed from the character pools.

### reqwest client Registration

`HibpClient` receives `reqwest` client via constructor injection. Register in Tauri state using `services.Addreqwest client<HibpClient trait, HibpClient>()` with a `User-Agent` header set to `DSPanel/{version}` and a 5-second timeout.
