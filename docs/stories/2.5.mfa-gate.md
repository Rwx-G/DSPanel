# Story 2.5: MFA Gate for Sensitive Actions

## Status: Draft

## Story

**As a** domain administrator,
**I want** sensitive actions to require MFA verification before execution,
**so that** compromised operator credentials alone cannot perform destructive changes.

## Acceptance Criteria

1. An `MfaService` trait interface supports TOTP verification and setup.
2. `TotpMfaService` implements RFC 6238 TOTP with HMAC-SHA1, 30-second time step, and 6-digit codes.
3. Sensitive actions (configurable per action in settings) require MFA verification before proceeding.
4. An `MfaDialog` blocks execution until a valid TOTP code is entered or the dialog is cancelled.
5. Failed MFA attempts are logged via `AuditService` trait and the action is blocked.
6. A setup wizard allows operators to configure their TOTP secret, displaying a QR code and backup codes.
7. The shared secret is stored encrypted on the local machine.
8. Unit tests cover TOTP generation/validation, dialog flow, and audit logging.

## Dev Notes

### TOTP Algorithm (RFC 6238)

The TOTP algorithm:
1. Compute time step: `T = floor((currentUnixTime - T0) / timeStep)` where `T0 = 0`, `timeStep = 30`
2. Convert `T` to 8-byte big-endian byte array
3. Compute `HMAC-SHA1(secret, T_bytes)`
4. Dynamic truncation: take 4 bytes starting at offset determined by last nibble of HMAC
5. Convert to 31-bit integer, then modulo `10^6` for 6-digit code
6. Left-pad with zeros if needed

### Clock Drift Tolerance

Accept codes from `T-1`, `T`, and `T+1` (90-second window) to accommodate minor clock drift between the operator's device and the server.

### Secret Storage

Use `Tauri secure storage.Protect()` (DPAPI) to encrypt the shared secret. Store the encrypted blob in `%LOCALAPPDATA%/DSPanel/mfa.dat`. This ties the secret to the current Windows user profile.

### QR Code Generation

Generate an `otpauth://` URI and render as a QR code data URI. Use a lightweight QR code library (e.g., `qrcode crate`) to generate the image in-memory without writing to disk.

### MfaGate Pattern

```typescript
public async Task<bool> ExecuteWithMfaAsync(Func<Task> action, string actionName)
{
    if (!_settings.RequiresMfa(actionName)) { await action(); return true; }
    var code = await _dialogService.ShowMfaDialogAsync();
    if (code == null) return false; // cancelled
    if (!await _mfaService.VerifyAsync(code)) { await _auditService.LogAsync("MfaFailed", ...); return false; }
    await _auditService.LogAsync("MfaVerified", ...);
    await action();
    return true;
}
```
