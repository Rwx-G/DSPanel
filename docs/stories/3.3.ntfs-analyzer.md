# Story 3.3: NTFS Permissions Analyzer

## Status: Draft

## Story

**As an** IT administrator,
**I want** a standalone NTFS permissions analyzer that shows inherited vs explicit ACEs with full group resolution,
**so that** I can audit folder permissions in depth without relying on external tools.

## Acceptance Criteria

1. A standalone "NTFS Analyzer" page is available in the navigation menu.
2. The operator enters a UNC path and the tool displays all ACEs with clear indication of inherited vs explicit.
3. For each ACE trustee that is a group, the full resolution chain is shown (group members, nested groups).
4. Recursive analysis allows scanning a folder and its subfolders to a configurable depth.
5. Deny rules are highlighted with a distinct visual style (e.g., red background, warning icon).
6. Conflicts between allow and deny at different levels are flagged.
7. Results can be exported to CSV and PDF.
8. Unit tests cover ACL resolution, group chain expansion, conflict detection, and export.

## Dev Notes

### Recursive Scanning

Recursive analysis uses `Directory.EnumerateDirectories()` with a depth counter. Each directory is processed independently so that an access denied error on one subdirectory does not abort the entire scan. Failed directories are logged and included in results with an error indicator.

### Group Chain Resolution

When an ACE trustee is a group SID, resolve it via `IDirectoryProvider.GetGroupMembersAsync(groupDn)`. For nested groups, recursively expand until only users remain. Display as a tree:

```
Trustee: IT-Department (group)
  - IT-Admins (group)
    - jsmith (user)
    - jdoe (user)
  - IT-Support (group)
    - bwilson (user)
```

Cache resolved groups in a `Dictionary<string, IList<DirectoryObject>>` to avoid duplicate LDAP queries during the same analysis session.

### Conflict Detection

A conflict exists when:
- The same trustee (or a group containing the same user) has both Allow and Deny ACEs for overlapping permissions
- A deny at a child folder overrides an allow inherited from a parent

Flag these with a warning icon and a tooltip explaining the conflict.

### PDF Export

Use a lightweight PDF library to generate formatted tables. Consider `QuestPDF` or `PdfSharp` for report generation. The PDF should include a header with the analysis date, scanned path, and operator name.
