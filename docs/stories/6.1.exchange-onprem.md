# Story 6.1: Exchange On-Prem Attributes (LDAP)

## Status: Done

## Story

**As a** HelpDesk operator,
**I want** to see Exchange on-premises mailbox information in the user detail view,
**so that** I can diagnose mail-related issues without switching to Exchange Management Console.

## Acceptance Criteria

1. When a user has `msExch*` LDAP attributes, an "Exchange" panel appears in the user detail view.
2. The panel displays: mailbox name, email aliases (proxyAddresses), forwarding target, recipient type, and delegate list.
3. The panel is hidden when the user has no Exchange attributes.
4. All Exchange fields are read-only (display only).
5. The Exchange attribute query is integrated into the existing `DirectoryProvider` trait user lookup flow.
6. Unit tests cover attribute detection, model mapping, and conditional panel visibility.

## Dev Notes

### LDAP Attributes

| LDAP Attribute               | Model Property                       | Notes                                                              |
| ---------------------------- | ------------------------------------ | ------------------------------------------------------------------ |
| `msExchMailboxGuid`          | `MailboxGuid`                        | Binary GUID, convert to string                                     |
| `msExchRecipientTypeDetails` | `RecipientType`                      | Numeric flag, map to string (1=UserMailbox, 2=LinkedMailbox, etc.) |
| `proxyAddresses`             | `PrimarySmtpAddress`, `EmailAliases` | Multi-valued; "SMTP:" prefix = primary, "smtp:" = alias            |
| `msExchDelegateListBL`       | `Delegates`                          | Multi-valued DN list of users with delegate access                 |
| `altRecipient`               | `ForwardingAddress`                  | DN of the forwarding target                                        |

### Recipient Type Mapping

| Value | Type             |
| ----- | ---------------- |
| 1     | UserMailbox      |
| 2     | LinkedMailbox    |
| 4     | SharedMailbox    |
| 8     | LegacyMailbox    |
| 16    | RoomMailbox      |
| 32    | EquipmentMailbox |
| 128   | MailUser         |
